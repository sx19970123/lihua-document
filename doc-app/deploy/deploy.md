# 打包部署

应用发行时将使用`.env.production`中的配置，请在打包前确认已按**生产环境**完成相关参数配置

## 打包步骤

1. 使用 **HBuilderX** 打开项目
2. 在顶部菜单中点击 **「发行」**
3. 根据需要选择对应的平台进行打包

![image-20251218111119994](./deploy.assets/image-20251218111119994.png)

## 注意事项

> 具体配置方式及发布流程，请参考各平台官方文档说明

- 不同平台在打包时对 **证书配置**、**签名方式**、**分发规则** 等要求各不相同
- 请根据实际发布的平台，提前准备并配置好相关证书
